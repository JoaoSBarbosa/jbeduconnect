# JBEduConnect

## Visão Geral

O JBEduConnect é uma plataforma de ensino desenvolvida em Java e Spring que visa facilitar a interação entre alunos e professores, oferecendo cursos, recursos educacionais, e um fórum de perguntas e respostas integrado. Administradores têm o controle total sobre o cadastro de cursos e turmas, garantindo uma solução flexível e personalizável para diversas necessidades educacionais.

## Funcionalidades Principais

- **Cadastro de Cursos e Turmas:** Administradores podem cadastrar cursos e turmas, personalizando o conteúdo conforme a necessidade de cada oferta.

- **Recursos Educacionais Diversificados:** Cada curso pode incluir trilhas de aprendizado, bônus, links externos e um fórum de perguntas e respostas, proporcionando uma experiência completa.

- **Tarefas Avaliadas:** Os alunos podem receber tarefas com peso, data de entrega e critérios de avaliação. Professores fornecem feedback e podem aceitar ou rejeitar as entregas.

- **Fórum de Perguntas e Respostas:** Uma plataforma de discussão robusta, permitindo filtrar, criar tópicos, responder, e votar nas melhores respostas.

- **Notificações:** Sistema de notificações para manter alunos e professores informados sobre eventos importantes.

## Requisitos

- Java 11 ou superior(LTS)
- Spring Boot 3.0 ou superior

## Instruções de Uso

1. Clone o repositório.

2. Configure o banco de dados MySQL.

3. Execute o aplicativo Spring Boot.
